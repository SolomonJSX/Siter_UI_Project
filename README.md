# Тестовое задание: Разработка адаптивного макета (Siter UI)

Профессиональная верстка многофункционального лендинга с использованием современного стека технологий. Проект выполнен в рамках технического задания с упором на чистый код, архитектуру БЭМ и отсутствие сторонних библиотек.

## 🚀 Технологии в проекте
*   **HTML5:** Семантическая верстка.
*   **SCSS:** Использование препроцессора, переменных, миксинов и модульной структуры.
*   **JavaScript Native:** Вся логика реализована на чистом JS без использования jQuery или других библиотек.
*   **Webpack 5:** Настроенная сборка проекта с оптимизацией ресурсов.
*   **БЭМ (Block Element Modifier):** Строгое соблюдение методологии именования классов.
*   **PostCSS & Autoprefixer:** Для обеспечения кроссбраузерности (последние 2 версии браузеров).

## 🛠 Реализованный функционал
1.  **Drill-down Menu:** Продвинутое мобильное меню с эффектом "перелистывания" уровней и кнопкой возврата.
2.  **Adaptive Design:** Полная адаптивность под три разрешения экрана:
    *   Мобильный (320px - 767px)
    *   Планшет (768px - 1023px)
    *   Десктоп (от 1024px)
3.  **Интерактивные элементы:**
    *   **Modal Window:** Модальное окно для заявок (открытие/закрытие, работа с оверлеем).
    *   **Native Slider:** Слайдер, написанный на чистом JS.
    *   **Accordion:** Интерактивный блок FAQ.
    *   **Logo Marquee:** Бесконечная бегущая строка с логотипами партнеров.
4.  **Кроссбраузерность:** Корректное отображение в Chrome, Firefox, Edge, Safari.
